# **App Name**: Lumina Vows

## Core Features:

- Romantic Poetry Generator: An AI-powered tool that generates bespoke romantic poems or welcome messages for Mohammed and Dina's guests to enjoy upon entry.
- Cinematic Hero Landing: An elegant introduction featuring a high-impact photo of the couple, smooth scroll effects, and their wedding date: 23 / 7 / 2026.
- Interactive Media Storyboard: A fluid grid of images and embedded love story videos with lightbox features for high-quality viewing.
- About the Couple Timeline: A beautifully typeset vertical layout chronicling Mohammed and Dina’s journey from meeting to engagement.
- Event Detail Dashboard: A clean, functional section displaying essential venue information, timings, and map integration for guests.
- Framer Motion Transitions: Orchestrated, smooth component-level animations for a high-end agency feel as the user scrolls.

## Style Guidelines:

- Primary color: Muted Antique Gold (#B39352) representing sophistication and timeless luxury.
- Background color: Creamy Linen (#F9F7F2), a highly desaturated version of the primary hue for a warm, soft atmosphere.
- Accent color: Dusky Rose (#BF7F7F), an analogous warm tone used sparingly for emotive emphasis and interaction points.
- Headlines: 'Playfair', a high-contrast serif for an elegant, high-end feel; Body: 'PT Sans', a humanist sans-serif for warmth and readability.
- Ultra-thin, minimalist gold-stroke icons used sparingly to mark logistical details like calendar dates and map locations.
- Single-page narrative structure with generous whitespace (margin-heavy) to let the typography and photography breathe.
- Staggered 'reveal' animations for text and image-fade transitions to evoke a slow-burning, romantic discovery process.
